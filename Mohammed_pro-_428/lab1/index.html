Контрольные вопросы

1. Что такое HTML?
HTML (HyperText Markup Language) — это язык гипертекстовой разметки, используемый для создания и структурирования разделов на веб-страницах.
2. Какова базовая структура верстки?
Базовая структура верстки включает в себя элементы html, head, и body. В head обычно помещают метаданные и заголовок страницы, а в body — содержимое веб-страницы.
3. Для чего используется элемент div?
Элемент div используется для группировки блоков контента и структурных элементов на веб-странице, что упрощает стилизацию и применение CSS.
4. Для чего используется элемент span?
Элемент span используется для группировки и стилизации частей текста внутри других элементов, не создавая при этом новых блоков.
5. Для чего используется элемент p?
Элемент p используется для разметки абзацев текста. Каждый абзац начинается с новой строки.
6. Для чего используется элемент head?
Элемент head содержит метаданные о веб-странице, такие как её заголовок, ссылки на стили, скрипты и другие инструкции, не видимые непосредственно на странице.
